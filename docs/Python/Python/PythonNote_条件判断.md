# 🐍 条件判断

> python的条件判断和 前端的JavaScript没太大不同  
> 区别仅仅就格式一点....  

### JavaScript👇
```js
if(x<10){
    console.log(x小于10)
}else{
    console.log(x大于10)
}
```
