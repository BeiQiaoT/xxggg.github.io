# 🧶 Yarn

学习前端的小伙伴一定不会对npm陌生，它是包管理工具  
很巧！Yarn也是！并且它的速度比npm快！

## 🧶Yarn 和🎒npm 能同时使用吗？
可以！二者都遵循`package.json`文件，完全不用担心兼容问题。  
你可以理解Yarn就是升级版的npm，能用Yarn就不要使用npm了~

## 🔨安装
```shell
npm i -g yarn
```

## 🔧使用  
1. yarn init 【初始化项目】
2. yarn add  【添加依赖】
3. yarn create 【创建Yarn工程】
4. yarn install 【安装项目的所有依赖项】
5. ......
