# ⚡ Vite

::: tip 官方网站
Vite：[https://vitejs.dev/](https://vitejs.dev/)
::: 
### ⚡ 环境依赖
-  node.js
-  Vite只能运行Vue3.x以上版本，不能运行Vue2.x
### ⚡ 安装Vite 创建项目
```sh
npm init vite
```
