# 🟥 CSS3 变形

### 旋转
```css
transform:rotate(45deg);
```
### 扭曲
```css
transform:skew(45deg);
```
### 缩放
```css
transform:scale(x,y);
```
### 位移
```css
transform:translate(x,y);
transform:translateX(x);
transform:translateY(y);
```
### 矩阵
```css
transform:matrix(a,b,c,d,e,f);
```
### 原点
```css
transform-origin:left top;
```
