# 🟥 CSS 零碎笔记

### 偏移自身一半
```css
.class{ transform: translate(-50%, -50%); }
```
### 去除a标签下划线
```css
.class{ text-decoration:none; }
```
### 去除列表的前缀符号
```css
.class{ list-style:none; }
```
### 字间距
```css
.class{ letter-spacing:20px; }
```
### 光标
```css
.class{ cursor:pointer; }
```
### 【权重】 !important
优先级最高
```css
.div_class{ color:blue  }
div{ color:red !important }
```
