# 🟥 CSS3 文本


### 对齐
```css
.demo{
    text-align:justify;
    text-justify:inter-word;
}
```
### 设置溢出文本用"省略标记"
```css
text-overflow : [clip 剪切掉] 或 [ellipsis 显示省略号]
```
### 强制文本在一行内显示
```css
white-space:nowrap
```
### 超过容器是否转行
```css
word-wrap:[normal 控制连续文本转行] 或 [break-word 内容在边界内换行]
```
###  文本阴影
```css
text-shadow:[X][Y][blur模糊程度][color]
```
