# ➕ Math数学方法

### 向上取整
```javascript
let xxg = Math.ceil(9.6)  //10
```
### 向下取整
```javascript
let xxg = Math.floor(9.6) //9
```
### 四舍五入
```javascript
let xxg = Math.round(9.5) //10
let xxg = Math.round(9.4) //9
```
### 随机数(0~1之间)
```javascript
let xxg = Math.random() // 随机数(0~1之间)

let xxgg = Math.random()*10 //(1~10之间)
```
